Pirateplayer-patch

Uppdateringar för Pirateplayer

Download Installer View on GitHub

Om PiratePlayer ger följande fel:

Ett fel uppstod!

eller bara visar en tom sida när du bläddrar bland play-programmen så kan du installera den här uppdateringen för att få det att fungera igen. I nuläget stöds svtplay.se, oppetarkiv.se, tv3play.se, tv4play.se, tv6play.se, tv8play.se och tv10play.se.

Installera genom att ladda hem och köra pirateplayer-patch-installer med knappen ovan. Du kan också läsa mer i README på GitHub. Rapportera gärna med GitHubs Issues-funktion om du hittar några fel.

svtplay-dl by spaam - media download engine used in this patch
