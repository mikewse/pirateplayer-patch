Problem med Pirateplayer?

Om Pirateplayer ger följande fel:

Ett fel uppstod!

eller bara visar en tom sida när du bläddrar bland play-programmen så kan du installera den här uppdateringen för att få det att fungera igen.

Kanaler

Installation

Installera genom att ladda hem pirateplayer-patch-install-filen med Download-knappen ovan och kör sedan denna.

Du kan också läsa mer i README på GitHub. Rapportera gärna med GitHubs Issues-funktion om du hittar några fel.
